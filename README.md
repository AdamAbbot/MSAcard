Usage Instructions
This browser extension enables card art replacement and UI button translation for Mobile Suit Arena. It works on all platforms that support browser extensions.
Tested environments:
Windows: Chrome
MAC: Safari
Linux: /
Android: Lemur Browser
IOS: Safari
Installation Guide
Open Chrome and go to the extensions page ([chrome://extensions/]). Enable Developer mode, then drag and drop the [MSAextension.zip] file to install.
Launch the game—no additional settings required. Card art and button text will be replaced automatically.
To use custom local card art, run [MSA.exe]. The program will load card files from your local [MSAcard] folder.
Additional Notes
This card pack only includes official GCG original card art—no alternate art or user-created cards are provided. For custom content, please use the local card replacement feature.Non-Chinese users should refer to the included [.docx] manual.
